# Inventory-Management-System
A complete CLI application for small businesses to track products, record multi-item sales with customers, and generate business reports.

Product Management: Add, update, delete, search, and list products with stock levels, buying/selling prices

Multi-item Sales: Record sales with multiple products per order, automatic stock deduction, customer tracking

Customer Management: Add customers, track repeat sales by name/phone

Advanced Reports:

Current inventory valuation (buying/selling)

Low stock alerts

Sales summary (all-time/today)

Top selling products

Sales by customer

Persistent Storage: JSON files for products, sales, customers (no database needed)

Robust Error Handling: Input validation, file corruption recovery, stock protection

Clean Architecture: Modular OOP design with separate concerns
